## 🔴 高优先级 - 补全已有框架

### 1. 分享功能补全

**现状**：依赖已配置（share_plus + screenshot），UI 入口存在，点击显示"分享功能开发中..."

**建议**：
- 截图分享：截取统计页热力图分享
- 文本分享：生成睡眠报告文本
- 分享卡片：设计精美的分享图片模板

**实现路径**：`services/share_service.dart` 已有框架，补完逻辑即可

### 2. 记录编辑/删除

**现状**：不支持修改或删除打卡记录

**建议**：
- 长按日期弹出编辑/删除选项
- 支持修改入睡时间
- 支持删除误打卡记录
- 添加操作确认弹窗

**用户价值**：基础容错功能，提升用户体验

### 3. 补卡功能

**现状**：不支持补录历史记录

**建议**：
- 允许点击过去的日期进行补卡
- 补卡记录添加标记（区分于正常打卡）
- 限制补卡时间范围（如只能补最近7天）

### 数据导出

```dart
// 导出 CSV
String exportToCSV(List<SleepRecord> records) {
  final buffer = StringBuffer();
  buffer.writeln('日期,入睡时间,级别');
  for (final record in records) {
    buffer.writeln('${record.date},${record.time},${record.level}');
  }
  return buffer.toString();
}
```

### 连续打卡计算

```dart
// 计算连续早睡天数
int calculateStreak(List<SleepRecord> records, String normalTime) {
  int streak = 0;
  // 按日期降序排列
  final sorted = records..sort((a, b) => b.date.compareTo(a.date));

  for (final record in sorted) {
    if (!record.isLate(normalTime)) {
      streak++;
    } else {
      break;
    }
  }
  return streak;
}
```
